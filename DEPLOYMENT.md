## 🚀 启用 GitHub Pages - 详细步骤

### Step 1: 访问仓库设置
1. 打开 GitHub 仓库：https://github.com/SUMSUNGRULER/stock-backtest
2. 点击顶部导航栏的 **Settings** 选项卡

### Step 2: 找到 Pages 设置
1. 在左侧菜单中，向下滚动找到 **Pages** 选项
2. 或直接访问：https://github.com/SUMSUNGRULER/stock-backtest/settings/pages

### Step 3: 配置 Pages
1. 在 **Source** 部分：
   - **Branch**: 选择 `main`
   - **Folder**: 选择 `/ (root)`
   - 点击 **Save**

2. GitHub 会自动部署你的网站（通常需要 1-2 分钟）

### Step 4: 查看部署状态
1. 在 Pages 设置页面，会看到绿色的成功消息：
   ```
   ✓ Your site is published at https://SUMSUNGRULER.github.io/stock-backtest/
   ```
